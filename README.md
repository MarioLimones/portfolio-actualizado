# Mario Limones Portfolio

Portfolio personal construido con React y Vite.

Nombre de repositorio recomendado para GitHub Pages:
`mariolimones.github.io`

## Desarrollo

```bash
npm install
npm run dev
```

## Publicacion

1. Crea el repositorio en GitHub con el nombre `mariolimones.github.io`.
2. Sube este contenido a la rama `main`.
3. En GitHub, activa Pages con `Source: GitHub Actions`.

El workflow de `.github/workflows/deploy.yml` se encarga del build y despliegue.
